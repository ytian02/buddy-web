# Roadmap

## Now

- 稳定 Buddy Web 的前端演示体验
- 完善版本日志和维护规范
- 提升 Buddy 的基础交互质量与一致性
- 固定 Buddy 主 mascot 的视觉方向和系统边界

## Next

- 增强 Buddy 的状态、情绪和反应表现
- 优化聊天回复策略，降低重复感
- 在固定 base rig 上补齐 skin、stage、accessory 的扩展能力
- 为未来自定义预留安全边界，而不是直接做自由编辑器
- 抽清未来接真实 API 的 adapter 边界
- 增加更系统的页面说明和运行文档

## Later

- 接入真实 LLM API
- 增加用户可配置的 companion 设置
- 扩展更多 species template 和 theme skin
- 设计成长驱动的 progression 机制
- 设计更完整的多页面信息结构
- 评估是否引入更自动化的发布和校验工具
